```bash

```

#### 2. Backend Setup

```bash
cd backend

# Install dependencies
uv sync

# Configure environment

# Edit .env with your Neon DATABASE_URL and JWT_SECRET_KEY
```

**Backend runs at**: `http://localhost:8000`

#### 3. Frontend Setup

```bash

# Edit .env.local with:
#   - DATABASE_URL (same as backend)
#   - BETTER_AUTH_SECRET (generate with: openssl rand -base64 32)
#   - JWT_SECRET_KEY (must match backend)
#   - NEXT_PUBLIC_API_URL=http://localhost:8000

# Run Better Auth migrations
npx @better-auth/cli migrate --config lib/auth/auth.ts

# Start frontend server
npm run dev
```

**Frontend runs at**: `http://localhost:3000`

#### 4. Access the Application

1. Open `http://localhost:3000`
2. Click **"Sign Up"** to create an account
3. Enter email and password (min 8 characters)
4. You'll be auto-logged in and redirected to the dashboard
5. Start creating tasks!
